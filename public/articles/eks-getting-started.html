https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html go to the macos section

4. set the kubeconfig

aws eks update-kubeconfig --region us-east-2 --name my-cluster

creates a kubernetes config file in 

~/.kube/config
and configures it for the cluster you just created in eks

check that it shows the information for your cluster

kubectl cluster-info

if you see: 

error: exec plugin: invalid apiVersion "client.authentication.k8s.io/v1alpha1"

you most likely have an old version of aws cli on your mac.

upgrade the version by following the getting-started-install sections above and then running the aws eks update-kubeconfig command again then kubectl cluster-info should work now we need to add nodes to our cluster now we need to add some pods to the cluster kubectl apply -f code/pyro-dashboard-api/fsim-pod.yml kubectl get pods
