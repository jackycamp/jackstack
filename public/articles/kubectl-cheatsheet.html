Kubectl Cheat Sheet

collection of kubectl commands that i use frequently

july 2023 boston

create or update resources in a Kubernetes cluster according to a configuration file.

kubectl apply -f my-spec.yml

show stdout and stderr for a pod

kubectl logs my-pod

show stdout and stderr for a running job

kubectl logs jobs/my-job

show stdout and stderr for a particular container of job

kubectl logs jobs/my-job -c my-container

display information about a resource (pod/job) in a Kubernetes cluster; includes configuration details as well as current status.

kubectl describe my-pod
